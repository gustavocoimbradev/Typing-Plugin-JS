Typing Plugin JS

A plugin to create a typewriter effect

Quick start

Download plugin files and import the .css and .js files in your project

Basic usage

This code will create a box with a text using the typewriter effect. Feel free to customize it, but keep the class .typing-js 
<div class="typing-js">

    <span typing-text="Hello, I'm typing this text"></span>

    <button class="js-decrease decrease">a-</button>
</div>

Additional attributes

There's some features you can control. To do it, just add the attributes below in the same tag you are using the classe .typing-js 
  • typing-time - writing time (ms) per letter typed (default: 40)
    • 

  • sleeping-time - interval time (ms) between the last character typed and the begining of the erasing (default: 2000)
    • 

  • deleting-time - erasing time (ms) per letter erased (default: 25)
    • 

  • interval-time - interval time (ms) between the texts typed (default: 250)
    • 

  • typing-loop - boolean value (1/0) to turn on or turn off the looping functionality (default: 1)
