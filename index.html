TypingJS

A plugin to create a typewriter effect

Quick start

Insert the code below into the <head> of your project 
<link rel="stylesheet" href="https://typingjs.vercel.app/css/plugin.css">


Insert the code below at the end of <body> of your project 
<script src="https://typingjs.vercel.app/js/plugin.js">

Basic usage

This code will create a box with a text using the typewriter effect. Feel free to customize it, but keep the class .typing-js 
<div class="typing-js">

    <span typing-text="Hello, I'm typing this text"></span>

</div>

Features

There's some features you can control. To do it, just set the values when you're calling TypingJS() 
<script>

    typingjs({
        selector: '.typing-js',
        typingTime: 40, // optional
        sleepingTime: 2000, // optional
        deletingTime: 25, // optional
        typingLoop: true, // optional
    })
</script>
